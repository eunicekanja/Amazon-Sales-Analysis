# Amazon-Sales-Analysis
Multi-year sales performance analysis using Power BI and DAX.

## Objective
Analyze Amazon sales data to evaluate revenue performance, identify category-level growth trends, and assess regional contribution patterns.

## Dataset
The dataset includes:
- Order date
- Sales revenue
- Product category
- Region / Country
- Quantity sold

## Key Analysis Performed
- Revenue trend analysis over time
- Year-over-Year (YoY) growth comparison
- Quarter-over-Quarter (QoQ) change analysis
- Category-level performance breakdown
- Regional sales distribution analysis

## Key Insights
- Revenue increased steadily throughout 2023 but experienced a slight decline of 2.01% in the final quarter. This decrease was primarily driven by reduced performance across the Asia, North America, and Europe regions.
- The Sports and Beauty categories also recorded declines, with most regions showing negative growth during the same period.

## Tools Used
- Power BI
- DAX (Time Intelligence)
- SQL (Data Extraction / Querying)
- Excel
